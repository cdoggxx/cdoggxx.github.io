Lab1

This lab shows how to use wireshark to analyze incoming and leaving messages on a network, specifically HTTP messages. This lab also teaches how use telnet for the same purpose. Next we develope a program in C to print statements to a web client. After, we use php to write a hello world program and an echo program. We then analyze the echo program and learn the difference between GET requests and POST requests.

Lab2

In this lab, we experimented with different html tags such as <div>, <a>, and <br>. We also learned how to insert images using <img> and we learned how to use forms with the <form> tag. We then learned how to use inline code using the <script> tag and we also learned how to write code in another .js file and include it in the HTML code to run. We were taught that using code from an outside source/link is not a good practice. Next, we explored how ajax and jquery can work together to make post and get requests use less lines of code and be easier to write. We also used fetch to return a promise.

Hackathon

In this hackathon, I hacked 6 levels of different websites to input a code snippet to show an alert. The websites used techniques such as filtering to remove the <script> tag and different techniques of inputting the data such as GET or POST. For levels 2-6, I have guessed what the source code may look like in order to make hacking it more difficult. Then, I modified my echo.php program and my waph-monaghan.html file to include input validation and encoding. Screenshots showing the GitHub merges are at the bottom. This lab taught me how easy it can be for hackers to hack a website if it isn’t secured properly. Now, I know how to secure my sites a little bit better from injection type attacks
